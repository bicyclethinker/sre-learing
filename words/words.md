

# **deprecated** /ˈdeprəkeɪtɪd/

> **已弃用的、不再推荐使用的。**

在计算机领域，它通常不是说“现在完全不能用”，而是：

- 目前可能还能正常工作；
- 官方已经不推荐继续使用；
- 将来的版本可能会删除；
- 应该逐渐换成新的命令、接口或写法。

| 单词          | 含义                           |
| ------------- | ------------------------------ |
| `deprecated`  | 还能用，但不推荐，未来可能删除 |
| `obsolete`    | 已经过时，基本被淘汰           |
| `removed`     | 已经被删除，不能再用           |
| `unsupported` | 不受官方支持                   |
