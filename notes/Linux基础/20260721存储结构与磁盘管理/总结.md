## 一、查看磁盘信息

```
lsblk
```

查看磁盘、分区、容量以及挂载位置。

```
lsblk -f
```

同时查看文件系统类型、UUID、挂载点。

```
blkid
```

查看设备的文件系统类型和 UUID。

```
df -Th
df -h
```

查看已经挂载的文件系统及其空间使用情况。

```
du -sh /目录
```

查看某个目录实际占用了多少空间。

------

## 二、创建和管理分区

### fdisk

```
fdisk /dev/sdb
```

主要用于交互式管理磁盘分区。

常用交互指令：

```
m   查看帮助
p   查看分区
n   创建分区
d   删除分区
w   保存并退出
q   不保存退出
```

### parted

```
parted /dev/sdb print
```

查看分区表。

```
parted /dev/sdb mklabel gpt
```

创建 GPT 分区表。

```
parted /dev/sdb mkpart primary 0% 100%
```

使用整个磁盘空间创建一个分区。

```
parted /dev/sdb rm 1
```

删除第 1 个分区。

```
partprobe
```

通知内核重新读取分区表。

> `parted` 操作通常实时生效，使用时要格外小心。

------

## 三、格式化文件系统

```
mkfs.ext4 /dev/sdb1
```

把分区格式化为 ext4 文件系统。

也可能遇到：

```
mkfs.xfs /dev/sdb1
```

格式化为 XFS 文件系统。

> `mkfs` 会破坏设备上原有的数据，必须先确认设备名称。

------

## 四、挂载和卸载

```
mkdir /mnt/data
mount /dev/sdb1 /mnt/data
```

把设备挂载到目录，之后通过该目录访问设备中的文件。

```
mount -o ro /dev/sdb1 /mnt/data
```

以只读方式挂载。

```
mount -o remount,ro /mnt/data
```

把已经挂载的文件系统重新挂载为只读。

```
umount /dev/sdb1
```

或者：

```
umount /mnt/data
```

卸载文件系统。

```
mount -a
```

按照 `/etc/fstab` 的配置执行挂载，常用于检查配置是否正确。

------

## 五、永久挂载

编辑：

```
vim /etc/fstab
```

常见格式：

```
设备或UUID  挂载目录  文件系统类型  挂载选项  备份  自检
```

例如：

```
UUID=xxxx-xxxx  /mnt/data  ext4  defaults  0  0
```

修改后必须测试：

```
mount -a
```

没有报错，再使用：

```
df -Th
```

确认挂载结果。

## 六、Swap 交换空间

```
mkswap /dev/sdb1
```

把分区格式化为 Swap。

```
swapon /dev/sdb1
```

启用 Swap。

```
swapoff /dev/sdb1
```

关闭 Swap。

```
free -h
```

查看物理内存和 Swap 使用情况。

## 七、软链接与硬链接

### 软链接

```
ln -s 原文件 链接文件
```

例如：

```
ln -s /etc/fstab fstab-link
```

软链接相当于快捷方式，原文件删除后链接失效。

### 硬链接

```
ln 原文件 链接文件
```

例如：

```
ln file1 file2
```

两个文件名指向同一个 inode，删除其中一个，另一个通常仍能使用。

```
ls -li
```

查看 inode 编号，判断两个文件是否为硬链接。

## 八、磁盘配额相关命令

这部分先做到“见到认识”即可：

```
quotacheck -avug
```

扫描文件系统并建立配额信息。

```
quotaon -avug
```

启用磁盘配额。

```
edquota -u user1
```

编辑用户配额。

```
edquota -p user1 -u user2
```

复制用户配额配置。

```
repquota -as
```

查看配额使用情况。

## 最核心的操作流程

把一块新磁盘投入使用，记住这条主线：

```
lsblk
  ↓
fdisk / parted 分区
  ↓
partprobe 刷新
  ↓
mkfs.ext4 格式化
  ↓
mkdir 创建挂载点
  ↓
mount 临时挂载
  ↓
df -Th 验证
  ↓
写入 /etc/fstab
  ↓
mount -a 测试
```

其中最需要熟练掌握的是：

```
lsblk
blkid
fdisk
parted
partprobe
mkfs.ext4
mount
umount
df -Th
du -sh 实际占用的存储空间 
mount -a
mkswap
swapon
free -h
ln
ln -s
```
