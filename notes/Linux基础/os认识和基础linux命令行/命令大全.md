| 时间戳   | 英文缩写 | 触发条件（什么时候变？）            | 核心特点                                   |
| -------- | -------- | ----------------------------------- | ------------------------------------------ |
| 访问时间 | Atime    | 读取文件内容（cat、less、打开查看） | 有“冷静期”：1天内反复查看只记1次，防刷屏   |
| 修改时间 | Mtime    | 修改文件内容（写入数据、编辑保存）  | 内容变了它就变，不管权限属主是谁           |
| 状态时间 | Ctime    | 修改文件属性（权限、属主）或内容    | 任何变动都触发（比 Mtime 更灵敏）          |
| 出生时间 | Birth    | 文件首次创建（touch、重定向生成）   | 一生只写一次，且很多老文件系统不支持显示 - |

### **文件查找**

find

见课件Linux基础05，文件查找小结



```shell
# 根据名称查找
find /root -name test.txt # 查找/root目录下所有名称为test.txt的文件
# 查找后处理
find / -name learvim.txt -exec ls -alh {} \;  # 查找/目录下所有名字为learvim.txt文件，并ls -alh查看信息
#{}代表占位， \是命令截至符
```
