# 这是sre学习笔记

# 每日的学习流程

## 白天


- 白天好好听课，把课上的实验进行复现，自己多做一些，加深理解
- 记录一些idea或者疑问到daily里面

## 晚上

- 整理笔记。前面部分的理论大部分可以直接复制课件。后面的实验部分喂给gpt，总结简洁的，好用的知识点
- 根据每天所学的知识点，自己自己在lab做实验，进行解释。

# 模块详情

## daily

- 每日记录学习要点，零感idea，复盘之类的杂项


## dashboard

- 记录当前的学习进度，3天一个周期，进行总结
- 9天一个大的周期，可以写篇日记啥的？呵呵。

## git_github

-  git版本管理的学习
- git的流程

## hidden

- 自己看的不提交的东西

## history

- 每天晚上结束前需要git当天的history
- 用chatgpt进行分析

## labs

- 每天都要针对当天的内容进行特定场景的训练

## notes

-  用时间加章节进行命名吧

- 学习的笔记
- 根据笔记，让gpt总结的核心命令，每天晚上笔记结束后和早上来的时候，复习一下

## prompt

- 负责每天发给chatgpt的提示词
- 顾名思义

## questions

记录一些奇思妙想，和可能重要的问题。有空就解决，没空就跳了吧= =

## words

记录一些计算机的单词，说不定会用上呢= =
