# 模块详情

## daily

- 每日记录学习要点，灵感idea，复盘

  > 每天的学习是双轨制的，即有自学的进度和跟老师的进度，daily根据需要可以记录两份或者一份 这个计划放弃。。。

## dashboard

- 记录当前的学习进度，3天一个周期，进行总结
- 9天一个大的周期，可以写篇日记啥的？呵呵。

## git_github

-  git版本管理的学习
- git的流程

## hidden

- 自己看的不提交的东西

## history

- 每天晚上结束前需要git当天的history
- 用chatgpt进行分析

## labs

- 每天都要针对当天的内容进行特定场景的训练

## notes

-  用时间加章节进行命令吧

- 学习的笔记
- 根据笔记，让gpt总结的核心命令，每天晚上笔记结束后和早上来的时候，背诵复习~

## prompt

- 负责每天发给chatgpt的提示词
- 顾名思义

# questions

记录一些奇思妙想，和可能重要的问题。有空就解决，没空就跳了吧= =
