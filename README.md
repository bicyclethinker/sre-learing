# 这是sre学习笔记

# 每日的学习流程

## 白天

- 根据需要，两份进度，上课听课和自学同时进行（实际上是自学不动了就听课吧。毕竟老师讲的学过一遍了。）
- 一些idea，灵感，有用的东西，随手丢到daily里面去啦。
- 一边学一边记录notes吧

## 晚上

- 重要的！！
- 更新daily模块，记录当天学习的内容，问题等等
- 最后半个小时：上传history，进行分析，让gpt生成几个特定场景的实验。方便第三第三天进行总结。

- labs真的很重要。

# 模块详情

## daily

- 每日记录学习要点，灵感idea，复盘
- 每天的学习是双轨制的，即有自学的进度和跟老师的进度，daily根据需要可以记录两份或者一份

## dashboard

- 记录当前的学习进度，3天一个周期，进行总结
- 9天一个大的周期，可以写篇日记啥的？呵呵。

## git_github

-  git版本管理的学习
- git的流程

## hidden

- 自己看的不提交的东西

## history

- 每天晚上结束前需要git当天的history
- 用chatgpt进行分析

## labs

- 每天都要针对当天的内容进行特定场景的训练

## notes

- 学习的笔记

## prompt

- 负责每天发给chatgpt的提示词
- 顾名思义

# questions

记录一些奇思妙想，和可能重要的问题。有空就解决，没空就跳了吧= =
